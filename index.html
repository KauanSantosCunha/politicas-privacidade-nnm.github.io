Política de Privacidade da Extensão "NNM - Nome na Mensagem"

Esta política de privacidade descreve como a extensão "NNM - Nome na Mensagem" lida com a privacidade dos dados do usuário.

1. Coleta e Tratamento de Dados

A extensão "NNM - Nome na Mensagem" não coleta, armazena, transmite ou compartilha qualquer tipo de dado de identificação pessoal. As informações inseridas pelo usuário (como o nome para a assinatura) são salvas localmente no navegador do usuário e não são enviadas para nenhum servidor externo.

2. Atividade e Conteúdo do Site

Para funcionar, a extensão interage com a página do WhatsApp Web. Isso inclui:

  • Detecção de cliques e teclas: A extensão monitora cliques no botão de envio e a tecla Enter para adicionar a assinatura automaticamente.
  • Manipulação de texto: O conteúdo da caixa de texto do WhatsApp é manipulado localmente para adicionar a assinatura antes do envio da mensagem.
  • Modificação do estilo: O estilo CSS da página é alterado para aplicar o efeito de desfoque nas conversas, um recurso de privacidade.

É importante ressaltar que todo esse processamento ocorre apenas no seu navegador, e a extensão não lê, armazena ou transmite o conteúdo das suas conversas para nenhum servidor externo.

3. Permissões

A extensão solicita as seguintes permissões para operar:

  • activeTab: Permite que a extensão se ative e funcione apenas quando o usuário estiver na aba do WhatsApp Web.
  • scripting: Essencial para injetar o código que permite interagir com a página, adicionar a assinatura e aplicar os efeitos visuais.
